API do BackEnd online

Este projeto e somente a API (sem pagina). A interface fica no projeto do FrontEnd. Endpoints disponiveis:

  • GET /api/teams — lista os times
  • POST /api/teams — cria um time
  • DELETE /api/teams?id=... — remove (header x-admin-token)
  • POST /api/scores — lanca nota (header x-admin-token)

Se /api/teams responder {"error":"no-db"}, falta definir a variavel DATABASE_URL (Neon) neste projeto na Vercel.
